# متجر العقيد | AL AKID

متجر إلكتروني جاهز (HTML/CSS/JS) بهوية بصرية فاخرة: أسود، ذهبي، فضي.

## 📁 محتويات المشروع
- `index.html` — الصفحة الرئيسية
- `style.css` — التصميم والألوان
- `script.js` — منطق المنتجات والسلة
- `assets/logo.png` — شعارك

---

## 🚀 خطوات رفع المتجر على GitHub ونشره مجانًا (بدون أي برمجة)

### 1) أنشئ حسابًا على GitHub
اذهب إلى **github.com** وأنشئ حسابًا مجانيًا إن لم يكن لديك واحد.

### 2) أنشئ مستودعًا (Repository) جديدًا
1. اضغط على زر **+** أعلى الصفحة ثم **New repository**.
2. اكتب اسمًا مثل: `al-akid-store`
3. اجعل المستودع **Public**.
4. اضغط **Create repository**.

### 3) ارفع ملفات المتجر
1. داخل صفحة المستودع الجديد، اضغط **uploading an existing file**.
2. اسحب مجلد المشروع بالكامل (أو الملفات: `index.html`, `style.css`, `script.js`, ومجلد `assets`) إلى الصفحة.
3. اضغط **Commit changes** بالأسفل لحفظ الرفع.

### 4) فعّل GitHub Pages (لنشر المتجر كموقع حقيقي)
1. من داخل المستودع، اذهب إلى **Settings**.
2. من القائمة الجانبية اختر **Pages**.
3. تحت **Branch** اختر `main` ثم اضغط **Save**.
4. انتظر دقيقة، وسيظهر لك رابط مثل:
   ```
   https://اسم-حسابك.github.io/al-akid-store/
   ```
5. افتح الرابط — متجرك أصبح مباشرًا على الإنترنت! 🎉

## ✏️ كيف تعدّل المنتجات لاحقًا؟

افتح ملف `script.js` وابحث عن `PRODUCTS` في الأعلى. كل منتج بهذا الشكل:

```js
{ id: 1, name: "بدلة كلاسيكية سوداء", cat: "رجالي", price: 890, oldPrice: 1100, icon: "🖤" }
```

- **name**: اسم المنتج
- **cat**: الفئة (رجالي / نسائي / إكسسوارات)
- **price**: السعر الحالي
- **oldPrice**: السعر قبل الخصم (اكتب `null` إن لم يوجد خصم)
- **icon**: رمز تعبيري مؤقت، أو استبدله بصورة حقيقية

### لإضافة صورة منتج حقيقية بدل الرمز:
1. ضع صورة المنتج داخل مجلد `assets` (مثلاً `assets/suit1.jpg`).
2. أضف للمنتج حقل صورة:
```js
{ id: 1, name: "بدلة كلاسيكية سوداء", cat: "رجالي", price: 890, oldPrice: 1100, image: "assets/suit1.jpg" }
```

بعد أي تعديل، احفظ الملف وارفعه مجددًا على GitHub (نفس خطوة الرفع بالأعلى) وسيتحدث الموقع تلقائيًا خلال دقائق.
